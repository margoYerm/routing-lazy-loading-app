Basics directives

  • NgClass [ngClass] Adds and removes CSS classes on an HTML element. See more
  • NgStyle [ngStyle] An attribute directive that updates styles for the containing HTML element. See more
  • NgFor *ngFor A structural directive that renders a template for each item in a collection. The directive is placed on an element, which becomes the parent of the cloned templates. See more
  • NgIf *ngIf="condition" A structural directive that conditionally includes a template based on the value of an expression coerced to Boolean. See more
  • NgSwitch [ngSwitch]="switch_expression" The [ngSwitch] directive on a container specifies an expression to match against. The expressions to match are provided by ngSwitchCase directives on views within the container. See more

How to create own directive

1. In cli type ng g d directives/name --skipTests

2. Create folder directives, in this project created ng-directives.. But in production project it's reserved for directives! In this folder create file name.directive.ts

In file name.directive.ts add 

@Directive(❴
  selector: '[appStyle]'
❵)

export class NameDirective ❴❵

Add this directive to declarations in this.module.ts
