Lesson 33. Passing params to the directive

This demo block handles Click on text element, mouse hover event. It also uses an object to pass style parameters

If we pass a string as a parameter to html, then we do not have to bind this parameter (wrapped in square brackets). If this is another data type (for example, an object, number, array), then it needs to be bind. 

appStyleParams="green" || [appStyleParams]="'green'"
[dStyles]="❴border: '1px dashed pink', 
            fontWeight: 'bold', 
            borderRadius: '10px'❵"
