What is Pipes

Lesson 20.

Just standard string. Syntax on double braces. In this case, Angular creates a new date object and calls its method .toString
{{ now }}

With pipe. Syntax: "now | date"
{{ now | date }}

With pipe. Syntax: "now | date: 'long'"
{{ now | date: 'long' }}

With pipe. Syntax: "now | date:'long' | uppercase"
{{ now | date:'long' | uppercase }}

With pipe. Syntax: "now | date:'long' | lowercase"
{{ now | date:'long' | lowercase }}
