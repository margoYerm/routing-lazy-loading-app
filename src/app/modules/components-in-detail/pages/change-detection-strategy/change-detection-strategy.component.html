Change Detection Strategy

Lesson 28. Change-detection-strategy works!


Every time there is a change in the app, Angular will perform ChangeDetectorRef on all the components. Whether it is a network request or user event, change detection will be performed by Angular.

When it comes to specific components, we can set ChangeDetectionRef strategy to OnPush. Angular will take this is a call to run Changedetectionstrategy on the components and their component chain alone.
