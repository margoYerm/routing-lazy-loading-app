Working with date

Lesson 37. How to working pipes with dates


See more

Demo how it works

{{ date }} //❴❴ date ❵❵

{{ date | date }} //❴❴ date | date ❵❵

{{ date | date:'fullDate' }} //❴❴ date | date:'fullDate' ❵❵

{{ date | date:'longDate' }} //❴❴ date | date:'longDate' ❵❵

{{ date | date:'long' }} //❴❴ date | date:'long' ❵❵

{{ date | date:'yyyy' }} //❴❴ date | date:'yyyy' ❵❵

{{ date | date:'MMMM dd yyyy' }} //❴❴ date | date:'MMMM dd yyyy' ❵❵

{{ date | date:'MMMM dd, yyyy, HH:mm:ss' }} //❴❴ date | date:'MMMM dd, yyyy, HH:mm:ss' ❵❵
