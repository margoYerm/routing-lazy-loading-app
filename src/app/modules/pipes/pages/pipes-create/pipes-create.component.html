How to create custom pipe

Lesson 41. Create pipe Lesson 42. Generate pipe with CLI

Create folder pipes in current module. And file name.pipe.ts into pipes folder.

In name.pipe.ts add import Pipe from @angular/core, add decorator @Pipe with name for using the pipe, add export class NamePipe implements PipeTransform. Add method transform to this class.

Add NamePipe to declarations in current module.

Or create pipe with CLI
ng g p .../pipes/name

Demo how it works
