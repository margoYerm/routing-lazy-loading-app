Working with numbers

Lesson 36. How to working pipes with numbers


Use pipes to transform strings, currency amounts, dates, and other data for display.

Pipes are simple functions to use in template expressions to accept an input value and return a transformed value. Pipes are useful because you can use them throughout your application, while only declaring each pipe once. For example, you would use a pipe to show a date as April 15, 1988 rather than the raw string format.

See more

Pipes do not require special connection. They are included in the base CommonModule, which is included in the BrowserModule.

Demo how it works

{{ e }} //❴❴ e ❵❵

{{ e | number }} //❴❴ e | number ❵❵

Passing pipes parameters (pipeName:string)
{{ e | number:'1.1-5' }} //❴❴ e | number:'1.1-5' ❵❵
{{ e | number:'3.1-2' }} //❴❴ e | number:'3.1-2' ❵❵
{{ e | number:'1.0-0' }} //❴❴ e | number:'1.0-0' ❵❵
